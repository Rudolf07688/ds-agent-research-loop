What I could not run here

Key decisions applied

  • F1 Vertex mode is driven from Settings via _configure_vertex() (single source of truth), not a mis-mapped env field.
  • F2 The live smoke script uses a persistent state dir so a second run exercises resume (zero seed calls) — making the SC-004 check valid.
  • Bounded agency: agents are built with tools=[] and transfers disallowed; asserted by an offline test.
